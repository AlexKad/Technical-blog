How to create global gitignore

  1. Set the path of the global gitignore file to ~/.gitignore_global.
    git config --global core.excludesfile ~/.gitignore

    In terms of implementation, that command adds the following entry
    excludesfile = ~/.gitignore_global

    You can find the file location by running this command in Terminal:
    git config --get core.excludesfile

  2. cd ~/
  3. touch .gitignore_global
  4. open -a TextEdit .gitignore_global

    Most common files to ignore: 

    # Ignore Mac system files
.DS_store

# Ignore node_modules folder
node_modules

# Ignore files related to API keys
.env

# Ignore SASS config files
.sass-cache
